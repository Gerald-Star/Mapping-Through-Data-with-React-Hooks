HTML FORM ELEMENT

HTML Form element is the container for different types of input elements such as text fields, checkboxes, radio buttons and submit buttons

The 'input element' is the most used type of the form element. Input element can be displayed in so many ways with type attribute.

Examples

  1. input type="text"
  2. input type="radio"
  3. input type="checkbox"
  4. input type="submit"
  5. input type="button
